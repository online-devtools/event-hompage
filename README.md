# Christmas 2025 Event Page

React Three Fiber와 Three.js를 사용한 인터랙티브 크리스마스 홈페이지입니다.

## 주요 기능

### 3D 비주얼 (고품질 그래픽)
- **현실적인 크리스마스 트리**: 5단 레이어, 64면 폴리곤, 자연스러운 색상 그라데이션
- **육각형 눈송이**: 실제 눈처럼 육각형 결정 모양, 회전하며 떨어지는 애니메이션
- **산타와 루돌프**: 빨간 코 루돌프와 함께 썰매를 타고 하늘을 비행
- **굴뚝 있는 집**: 따뜻한 창문 빛, 굴뚝에서 피어오르는 연기 파티클
- **선물 상자**: 빛나는 리본과 애니메이션이 적용된 선물들
- **별이 빛나는 배경**: 2500개의 별로 이루어진 우주 배경
- **향상된 조명**: 환경 맵, 그림자(2048x2048), 다중 광원
- **고급 머티리얼**: 메탈릭, 러프니스, 이미시브 효과 적용

### 인터랙티브 기능
- **크리스마스 카드 작성**: 따뜻한 메시지를 담은 카드를 3D 공간에 띄울 수 있습니다
- **새해 소원 빌기**: 트리의 장식 구슬에 2025년 소원을 적을 수 있습니다
- **2025년 신년 운세**: 10가지 운세 중 랜덤으로 뽑을 수 있는 재미있는 기능
- **클릭 인터랙션**: 카드와 구슬을 클릭하여 작성된 메시지를 확인할 수 있습니다
- **로컬 저장**: 작성한 카드와 방명록이 브라우저 `localStorage` 에 저장됩니다
- **마우스 컨트롤**: 드래그로 화면 회전, 휠로 줌 인/아웃 가능

### 반응형 디자인
- 모바일, 태블릿, 데스크톱 모두 완벽 지원
- 터치 제스처 지원 (모바일)

## 컴포넌트 구조

### 3D 컴포넌트
- `ChristmasTree.tsx`: 현실적인 5단 트리 (64-sided polygons, 자연스러운 색상)
- `Snowfall.tsx`: 육각형 눈송이 파티클 (150개, ExtrudeGeometry 기반)
- `SantaSleigh.tsx`: 산타와 루돌프 썰매 애니메이션 (원형 경로 비행)
- `ChristmasHouse.tsx`: 굴뚝 있는 집 (창문 빛, 연기 파티클 애니메이션)
- `Gifts.tsx`: 고급 머티리얼의 선물 상자 그룹
- `WishCard.tsx`: 3D 카드 컴포넌트 (클릭 가능, 검은 텍스트 + 흰색 아웃라인)
- `InteractiveOrnament.tsx`: 클릭 가능한 트리 장식 구슬 (호버 효과)
- `ChristmasScene.tsx`: 전체 3D 씬 관리 (고품질 렌더링 설정)

### UI 컴포넌트
- `CardModal.tsx`: 카드/소원 작성 모달 (검은색 텍스트 입력)
- `FortuneModal.tsx`: 신년 운세 뽑기 모달 (10가지 운세, 애니메이션 효과)
- `App.tsx`: 루트 컴포넌트 (3D 씬 + UI + 상태 관리)

## 사용 방법

### 기본 조작
- **마우스 드래그**: 화면을 회전시켜 다양한 각도에서 트리 감상
- **마우스 휠**: 줌 인/아웃 (최소 3, 최대 10 거리)
- **모바일**: 터치로 드래그 및 핀치 줌 가능

### 인터랙티브 기능
1. **카드 작성하기**
   - 우측 하단의 "💌 카드 작성" 버튼 클릭
   - 크리스마스 메시지와 이름 입력
   - 카드가 3D 공간에 원형으로 배치됩니다
   - 카드 클릭 시 전체 메시지 확인 가능

2. **소원 빌기**
   - 우측 하단의 "⭐ 소원 빌기" 버튼 클릭
   - 2025년 새해 소원 입력
   - 트리의 장식 구슬이 금색으로 빛나며 소원이 저장됩니다
   - 구슬에 마우스를 올리면 소원 미리보기
   - 구슬 클릭 시 전체 소원 확인

3. **운세 뽑기**
   - 우측 하단의 "🎴 운세 뽑기" 버튼 클릭
   - 금색 상자에서 랜덤으로 운세 추첨
   - 10가지 운세 중 하나: 대길, 길, 중길, 소길, 말길, 반길, 희망의 해, 성장의 해, 행복의 해, 풍요의 해
   - 각 운세마다 다른 색상과 이모지 제공
   - 다시 뽑기 가능

4. **데이터 관리**
   - 카드·방명록은 브라우저 `localStorage` 에 저장되어 같은 기기/브라우저에서는 새로고침 뒤에도 유지됩니다
   - 다른 사람과 데이터가 공유되지는 않으므로, 새로운 브라우저나 기기에서는 항상 초기 상태로 시작합니다
   - 저장된 데이터를 지우고 싶다면 브라우저 저장소를 비우거나 `localStorage.removeItem('christmas-cards')` 등으로 정리하세요
